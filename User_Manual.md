## Getting Started

### Prerequisites
- A device with a web browser (tablet, laptop, or desktop)
- Internet connection
- For drawing questions: A touchscreen device works best, but mouse input is also supported
- For photo questions: A device with a camera or the ability to upload images

### Accessing the App
1. Open your web browser
2. Navigate to the app URL provided by your school or organization
3. You'll land on the Setup page to create your child's profile

---

## Setting Up Your Child's Profile

When you first open the app:

1. **Enter Child's Name**: Type your child's name in the input field (e.g., "Alex", "Sam", or any name you prefer)
2. **Select Grade Level**: Choose the appropriate grade from the dropdown menu
   - Grades typically range from Grade 1 to Grade 8
   - Select the grade that matches your child's current math curriculum
3. **Click "Start Learning"**: This creates the profile and takes you to the main dashboard

**Note**: The name you enter is stored locally for your session and is used to personalize the learning experience.

## Creating Assignments

### Step 1: Choose a Topic
From the dashboard, you'll see a list of math topics organized by grade level:
- Number Sense
- Patterns and Algebra
- Geometry
- Measurement
- Data Management
- And more...

Click on any topic to explore sub-topics.

### Step 2: Generate Exercises
1. Select a specific sub-topic (e.g., "Nets of Solids" under Geometry)
2. Choose the number of questions (typically 5-10)
3. Click **"Generate Assignment"**
4. Wait a few moments while AI creates personalized exercises

### Step 3: Start the Assignment
Once generated:
- Review the assignment summary
- Click **"Start Assignment"** to begin
- The app will guide your child through each question one at a time

## Sololving Exercises

### The Exercise Interface

Each exercise screen has these components:

#### Question Card (Top)
- Displays the question prompt
- May include instructional diagrams or reference images
- Read the question carefully before answering

#### Input Mode Selector (Middle)
Depending on the question type, you'll see different options:

**For Visual Multiple-Choice Questions:**
- Four option cards arranged in a 2×2 grid
- Each card shows a diagram labeled A, B, C, or D
- Tap/click an option to select it (highlighted with a teal ring)

**For Drawing Questions:**
- A digital canvas with drawing tools
- Tools include: pen, eraser, clear canvas
- Draw your answer directly on the canvas

**For Typing Questions:**
- A text input box
- Type your numerical or written answer
- Use the on-screen keyboard or physical keyboard

**For Photo Questions:**
- A camera button to take a photo
- An upload button to select an existing image
- Follow the question instructions to show your work

#### Action Buttons (Bottom)
- **Hint**: Get a helpful clue without giving away the answer
- **Save Answer**: Submit your response
- **Next Question**: Automatically appears after saving

### How to Answer Different Question Types

#### Visual Multiple-Choice (New Feature!)
These questions show four diagram options:

1. **Read the question** carefully (e.g., "Which of these is a net of a cube?")
2. **Examine each option** (A, B, C, D)
3. **Tap the card** with your chosen answer
   - Selected option gets a teal highlight ring
4. **Click "Save Answer"**
5. The app confirms: "You selected option [X] — saved ✓"

**Example**: For cube net questions, remember:
- A valid cube net has exactly 6 squares
- The squares must be connected edge-to-edge
- There are 11 possible valid nets of a cube

#### Drawing Questions
1. Use your finger (touchscreen) or mouse to draw
2. Draw clearly and label if needed
3. Use the eraser to fix mistakes
4. Click "Clear" to start over if needed
5. Click "Save Answer" when done

#### Typing Questions
1. Click in the text box
2. Type your answer (numbers, words, or short expressions)
3. Double-check your entry
4. Click "Save Answer"

#### Photo Questions
1. Click the camera icon to take a new photo, OR
2. Click the upload icon to select an existing image
3. Position your work clearly in the frame
4. Review the preview
5. Click "Save Answer"

### Using Hints
- Click the **Hint** button anytime during a question
- Hints provide guidance without revealing the answer
- Example hint: "Think about how many faces a cube has"
- Use hints when stuck, but try to solve independently first

### Navigating Between Questions
- After saving an answer, the app may auto-advance to the next question
- Or click **Next Question** manually
- A progress indicator shows how many questions remain

## Viewing Results

After completing all questions in an assignment:

### Results Summary Page
You'll see:
- Total score (e.g., "7 out of 10 correct")
- Breakdown by question
- Time spent on the assignment

### Per-Question Feedback

For each question, the results show:

#### Correct Answers
- Green checkmark ✓
- "Correct!" message
- For MCQ: Shows the option grid with correct answer highlighted in green

#### Incorrect Answers
- Red X mark ✗
- "Your answer: [X]" vs "Correct answer: [Y]"
- For MCQ: Shows both selections with:
  - Green border on the correct option
  - Red border on your selected (wrong) option
- Explanatory feedback when available

#### Review Mode
- Scroll through all questions
- Compare your answers with correct ones
- Learn from mistakes

## Question Types

The app supports four main question formats:

### 1. Visual Multiple-Choice (MCQ with Diagrams)
- **What it is**: Four diagram options to choose from
- **Examples**:
  - "Which shape is a hexagon?"
  - "Which is a net of a cube?"
  - "Which graph represents this data?"
- **How to answer**: Tap the correct option card (A, B, C, or D)
- **Evaluation**: Instant letter-matching (deterministic)

### 2. Drawing Questions
- **What it is**: Freehand drawing on a digital canvas
- **Examples**:
  - "Draw a rectangle with area 12 square units"
  - "Sketch the reflection of this shape"
  - "Draw lines of symmetry"
- **How to answer**: Use drawing tools to create your response
- **Evaluation**: AI analyzes the drawing for correctness

### 3. Typing Questions
- **What it is**: Text or numerical input
- **Examples**:
  - "What is 7 × 8?"
  - "Solve for x: 2x + 5 = 15"
  - "Write the next number in the pattern"
- **How to answer**: Type your answer in the text box
- **Evaluation**: Exact match or semantic analysis

### 4. Photo Questions
- **What it is**: Upload or capture an image of handwritten work
- **Examples**:
  - "Show your work for this multi-step problem"
  - "Draw a bar model and photograph it"
  - "Complete the table and upload a photo"
- **How to answer**: Take a photo or upload an image
- **Evaluation**: OCR and AI analysis of the image content

## Tips for Parents

### Creating Effective Practice Sessions
1. **Choose appropriate difficulty**: Match topics to your child's current level
2. **Start small**: Begin with 5-question assignments to build confidence
3. **Review together**: Go through results and discuss mistakes
4. **Encourage hints**: Teach kids that using hints is part of learning
5. **Celebrate progress**: Focus on improvement, not just perfect scores

### Supporting Your Child
- **Read questions aloud** for younger children
- **Encourage scratch paper** for working out problems
- **Discuss wrong answers** to understand misconceptions
- **Use real-world connections** (e.g., measure objects for geometry)
- **Set regular practice times** (15-20 minutes daily is ideal)

### Understanding the Technology
- **AI-generated questions**: Each exercise is unique, created based on the topic
- **Instant feedback**: Most questions are evaluated immediately
- **Adaptive learning**: The system can adjust difficulty based on performance
- **Visual learning**: Diagram-based questions help develop spatial reasoning

### Privacy & Safety
- Child names are stored locally for personalization only
- No personal data is shared externally
- Photos are processed securely and not stored permanently
- The app complies with educational privacy standards

## Troubleshooting

### Common Issues

#### "Drawing isn't working"
- Ensure you're using a compatible browser (Chrome, Safari, Firefox, Edge)
- Try clearing browser cache
- On touchscreens, make sure touch input is enabled
- On desktop, use mouse clicks to draw

#### "Camera won't activate"
- Grant camera permissions when prompted
- Check that no other app is using the camera
- Try the upload option instead of live camera
- Ensure your browser supports camera access

#### "Assignment won't generate"
- Check your internet connection
- Wait 10-15 seconds for AI generation
- Try refreshing the page and regenerating
- If persistent, try a different topic

#### "Answer won't save"
- Ensure you've completed the input (drawn something, typed text, etc.)
- Wait for the loading indicator to finish
- Check internet connection
- Try refreshing and re-attempting the question

#### "Results page shows incorrect evaluation"
- Review the question and your answer carefully
- Some questions allow multiple correct forms
- Contact support if you believe there's a systematic error
- Remember: MCQ questions are exact-match (letter comparison)

### Browser Compatibility
The app works best on:
- Google Chrome (recommended)
- Safari (iOS/macOS)
- Microsoft Edge
- Firefox

Minimum requirements:
- Modern browser (2020 or later)
- JavaScript enabled
- Screen resolution: 1024×768 or higher

### Getting Help
If you encounter issues not covered here:
1. Refresh the page
2. Clear browser cache and cookies
3. Try a different browser
4. Check your internet connection
5. Contact your school's IT support or the app administrator

## Glossary

- **Assignment**: A set of generated exercises on a specific topic
- **Exercise**: A single question or problem to solve
- **Visual MCQ**: Multiple-choice questions with diagram options
- **Net**: A 2D pattern that folds into a 3D shape (e.g., cube net)
- **OCR**: Optical Character Recognition (reading text from images)
- **Hint**: A clue to help solve a problem without giving the answer
- **Structured Answer**: The format used to evaluate responses (text, drawing, photo, or MCQ selection)
